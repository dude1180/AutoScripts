# AutoScripts
A collection of Python-based automation and macro scripts for various games and repetitive tasks

> ⚠️ **CRITICAL REQUIREMENT:** All scripts in this repository that interact with emulators or system-level windows **MUST be executed as Administrator**. Otherwise, Windows will block the simulated inputs. Note that these scripts are exclusively designed for Windows PCs and Android emulators (such as MSI APP PLAYER).

## Prerequisites
Before running the scripts, make sure you have Python installed and the required libraries:

```bash
pip install pyautogui 
pip install mss
pip install keyboard
pip install easyocr 
pip install mss
pip install numpy
pip install opencv-python
