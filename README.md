# AutoScripts
A collection of Python-based automation and macro scripts for various games and repetitive tasks

> ⚠️ **CRITICAL REQUIREMENT:** All scripts in this repository that interact with emulators or system-level windows **MUST be executed as Administrator**. Otherwise, Windows will block the simulated inputs.
